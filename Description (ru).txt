* Основное:
W - вперёд (ускорение/вверх)
S - назад  (торможение/вниз)
A - поворот против часовой стрелки
D - поворот по часовой стрелки
Space - ускорение
Enter - меню

* Комбинации:
Вперёд + Space - турбонаддув
Назад + Space  - ручной тормоз

* Отладка:
I - скрол карты вверх
K - скрол карты вниз
J - скрол карты влево
L - скрол карты вправо

* Реализованно:
v 0.2
- добавлено меню настроек (Enter)
- переработано поведение/управление машинкой (упрощённая физика)
- добавлена коллизия с объектами (динамическими/статическими)

v 0.1
- условно бесконечный мир (поверхность)
  размерность генерации мира 2^32*2^32
- генерация происходит из seed (слово) + частота (скейл мира)
- объекты (декали/коллизии) формируются всегда на своём месте
- сортировка объектов по оси Y

* Известные баги:
v 0.2 
- [x] при не понятных действиях возможна порча спрайтов тайлопар (сдвиговых)
- [x] нет привязки скорости машинки к прерываниям (на разных частотах разное поведение)
- [x] декали проходят тест сортировки по Y

v 0.1
- [ ] нет коллизий у объектов с коллизией
- [ ] декали проходят тест сортировки по Y
- [ ] нет привязки скорости машинки к прерываниям (на разных частотах разное поведение)
- [ ] при не понятных действиях возможна порча спрайтов тайлопар (сдвиговых)
- [ ] спавн нескольких объектов (декали/коллизии) на одном месте

* Примечание:
v 0.2
- из-за особенностей движка, в силу плавности картинки на 3.5МГц, возможно
  небольшое расхождение по скорости разгона/перемещения, нежели на 7МГц+
